Register row R-34, adjudicated in item-0072 as D1 per rulebook RB-05 and ADR-049 (1). The target forms are already modelled: AbsPath carries the socket path, Authority carries the tcp host:port. The scheme prefix is the type switch, not a missing type, so no contract definition is added -- the engine address never reaches the wire (contract/endpoint/engine.cue attests type and cert identity, never an address).
