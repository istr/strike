ADR-049 (5). tools/genenums recovers disjunct values only from named definitions, so an inline disjunction generates a bare string field whose admissible values exist nowhere in Go. Five sites. Struct unions whose arms carry a single concrete discriminator literal are structurally different and out of scope. Wire-neutral: the CUE definition name and the Go type name are not the JSON field name.
