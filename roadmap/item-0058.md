The ratified discriminator: CUE-expressible AND crosses a package boundary (a necessary export is typed surface) -> must be CUE-generated; package-local OR behavioral -> hand-written; for disjunctions the arms are generated and only the interface plus the dispatching UnmarshalJSON stay hand-written. Test-driven: built after the attest-gengotypes spike, then the cue-coverage migrations make it green. It replaces any hand-maintained exemption list -- the scan IS the gate, so nothing data-shaped can hide behind 'it is a service object'. Distinct from the layer-direction lint (item-0016): that enforces tier direction, this enforces migrate-to-CUE coverage. Intended execution position: after the spike, before the migration clusters, so the migrations are driven by a red gate.
