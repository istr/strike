Push-target grammar. endpoint reuses endpoint.#TLS (authority plus #Trust anchor). #OCIName is anchored ^...$ because CUE =~ is a partial match, so the unanchored distribution-spec regex would accept ubuntu:latest and ubuntu@sha256. No tag, no digest: the digest is the push result the control plane computes. IPv6 literals are parked in item-0101. Static and readable, so predict-and-gate.
